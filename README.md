## 操作方法
- 上部設定バーで操作します
  - マイク設定
    - マイクがいくつ接続されていても、必ず10つ表示されます
    - マイクに音を入れながら、何番のマイクが正しいかチェックしてください
    - 該当番号にマイクが接続されていない場合、エラーが表示されます
  - 口パク設定
    - 自動：勝手にずっと口パクします
    - マイク：マイクの音量を見て、口パクする／しないを切り替えます
    - 開く：常に口を開きっぱなしにします
    - 閉じる：常に口を閉じっぱなしにします
